集群管理页面可统一管理通过部署云应用所构建的集群。

说明

本文简要介绍集群管理提供的相关基础通用操作，更多关于集群的详细介绍及操作说明，请查看各应用或产品对应的文档。

例如：管理 MySQL Plus 集群，请查看 MySQL Plus 文档。

集群管理入口

  1. 登录管理控制台。

  2. 选择产品与服务> AppCenter 控制台 > 集群管理，进入集群管理页面。

启停集群

支持关闭、启动及重启集群。

  • 批量操作：在集群管理页面，勾选已部署的集群（可多选），点击集群列表上方的启动或关闭，对集群进行快速的启动、关闭。

  • 单独操作：

    • 在目标集群的操作列，点击关闭、启动或重启，对该集群进行启动、关闭或重启。

    • 右键点击目标集群，在弹出的菜单列表中，点击关闭、启动或重启，对该集群进行启动、关闭或重启。

集群设置

支持修改集群名称与描述信息以及升级集群。

  1. 在集群管理页面，右键点击目标集群，弹出菜单列表，选择集群设置。

  2. 通过点击子菜单，进行如下设置操作：

    • 修改信息：修改集群名称及描述信息。

    • 版本升级：升级集群到新版本。

      说明

      当集群有新版本时，才可选择升级。

集群运维

支持集群扩容、创建集群备份、修改备份策略以及更改云服务器类型。

  1. 在集群管理页面，右键点击目标集群，弹出菜单列表，选择运维管理。

  2. 通过点击子菜单，进行如下运维操作：

    • 集群扩容：通过升级集群节点配置（如：CPU、内存、硬盘），进行集群扩容。

    • 创建备份：创建集群数据备份。备份后，可共享备份给其他账户或从备份创建集群。

    • 修改备份策略：设置是否开启自动备份，以及备份链个数和长度。

集群费用

支持查看费用详情和修改计费模式。

说明

需要云平台支持计费功能，否则费用管理操作对用户不可见。

  1. 在集群管理页面，右键点击目标集群，弹出菜单列表，选择费用管理。

  2. 通过点击子菜单，进行如下费用管理操作：

    • 费用信息：查看集群及备份的费用详情。

    • 修改计费模式：按需计费及包年包月模式切换。

      说明

      当集群只支持一种计费模式时，无法修改计费模式。

集群标签

支持为集群绑定标签，以便通过标签识别及筛选集群。

  • 绑定标签

    • 批量操作：在集群管理页面，勾选目标集群（可多选），点击集群列表上方的更多操作 > 绑定标签，为所选集群绑定一个或多个标签。

    • 单独操作：在集群管理页面，右键点击目标集群，弹出菜单列表，选择标签 > 绑定，为当前集群绑定一个或多个标签。

  • 解绑标签

    在集群管理页面，右键点击目标集群，弹出菜单列表，选择标签 > 解绑，将标签与该集群解除绑定。

  • 标签过滤

    在集群列表右上方，点击标签，选择需要过滤的标签（可多选），即可筛选出绑定所选标签的集群。

集群与项目

支持将集群添加到项目，以便通过项目维度查看和管理集群资源。

说明

一个集群资源只能被添加到一个项目，一个项目可添加多个集群资源。

  • 添加到项目

    • 批量操作：在集群管理页面，勾选目标集群（可多选），点击集群列表上方的更多操作 > 添加到项目，将所选集群添加到项目。

    • 单独操作：在集群管理页面，右键点击目标集群，弹出菜单列表，选择添加到项目，选择需要添加的项目，点击确定。

  • 离开项目

    在集群管理页面，右键点击目标集群，弹出菜单列表，选择离开项目，在提示框中点击确定。

删除集群

  • 批量操作：在集群管理页面，勾选需要删除的集群，点击集群列表上方的更多操作 > 批量删除，删除所选集群。

  • 单独操作：右键点击目标集群，在弹出的菜单列表中，点击删除，删除当前集群。

说明

若要确保集群一次性成功删除，删除时请勾选强制删除。

集群详情

您可以查看集群详细信息，包含集群基本信息、服务端口、租赁信息和操作日志。

  1. 在集群管理页面，点击集群名称/ID 列的集群 ID，进入该集群详情页面。

    如 Redis Standalone 集群，详情页面如下：

  2. 在集群详情页面左侧，您可以查看集群基本属性、租赁信息、操作日志、服务端口等信息。

    • 基本属性：包含集群的 ID、状态、名称、标签、描述、应用名称、版本、节点数量、私有网络、授权提供商、创建时间等。

    • 租赁信息：可查看具体的租赁时间及计费信息。需要集群支持计费功能，否则，不展示该信息。

    • 服务端口：集群应用部署后，若有端口的配置，可查看相关端口信息。若无端口的配置相关，则不展示。

    • 操作日志：点击查看集群具体操作信息。

更多操作

进入集群详情页面后，在页面右侧区域，可进行更高级的集群管控操作，如下图为 Redis Standalone 的操作界面。

不同集群支持的管理功能有所差异，通常有以下功能：

  • 节点管理：支持新增、修改、删除集群节点，以及节点监控信息查看。

  • 配置参数：支持修改集群配置参数。

  • 告警：支持配置集群告警指标及告警策略。

  • 备份：支持创建备份、查看集群备份链信息、删除备份、从备份创建集群。

更多操作，请参见各应用对应的文档。
