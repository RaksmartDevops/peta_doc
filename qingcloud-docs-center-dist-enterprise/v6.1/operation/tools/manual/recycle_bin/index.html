回收站

本文以删除一台云服务器然后恢复的过程为例，介绍回收站功能。

删除一台云服务器

在控制台，选择计算 > 云服务器页面，选中要删除的云服务器，然后点击删除。

找到回收站中被删除的云服务器

选择运维与管理 > 回收站页面，查找刚刚删除的云服务器。

恢复被删除的云服务器

选择运维与管理 > 回收站页面，选中刚刚删除的云服务器，然后点击恢复。

注意

回收站只会保留被删除的资源 2 个小时。2 个小时后，资源会被彻底销毁，不可恢复。
