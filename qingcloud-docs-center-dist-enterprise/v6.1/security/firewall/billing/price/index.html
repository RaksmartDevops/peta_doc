本节介绍云防火墙服务的计费说明。

计费模式

云防火墙提供包年包月计费模式，购买服务时，您需要预先支付购买时长内的费用。购买时长越长，优惠力度越大。

产品定价

云防火墙分为高级版和旗舰版两种版本，不同的版本支持的可防护的带宽跟公网 IP 不同，价格也不同。

  • 额外费用：扩展带宽 50元/Mbps/月，扩展公网 IP 50元/个/月。

到期与续费

若您需要持续使用云防火墙服务，您需要在服务到期前进行手动续费，到期后若未续费，将无法继续使用云防火墙进行资产防护。

云防火墙到期后，如果没有按时续费，云平台会提供 7 天的保留期，保留期内，您可随时续费使服务恢复正常，但不可进行其他操作。过期超过 7 天，云防火墙相关配置数据将被删除。
