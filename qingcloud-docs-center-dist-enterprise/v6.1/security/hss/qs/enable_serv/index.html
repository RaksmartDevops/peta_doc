步骤一：购买主机安全服务

更新时间：2024-03-07 19:06:53

首次使用主机安全服务之前，您需要先购买（或开通）主机安全服务，获得云服务器防护授权及防篡改授权。

背景信息

本文档中所有操作指导以主机安全服务开启计费对接的情况下进行介绍，若您当前环境未启用计费功能，则不支持购买、支付、金额计算等费用结算功能，界面中“购买”、“立即支付”等描述可能被替换成“开通”、“确定”（以实际环境为准），但产品本身操作使用并无区别，您仍可参考本文进行相关操作。

购买须知

  • 您可一次购买多个授权，一个授权只能绑定一个云服务器。

  • 购买授权后，须在需要防护的云服务器上安装主机安全 Agent，才能将授权绑定到云服务器上，云服务器才能获得相应的安全防护能力。

前提条件

  • 已注册云平台账号，且已完成实名认证。

  • 账号余额大于 0。

操作步骤

  1. 登录管理控制台。

  2. 在顶部菜单栏，选择产品与服务 > 安全服务 > 主机安全，进入主机安全服务购买页面。

  3. 点击立即购买，进入配置信息页面。

  4. 配置购买信息。

    参数 说明

    选择管理中心

    管理中心用于接收 Agent 上报的主机信息并进行分析处理，每个管理中心对应一个控制台。

    每个管理中心的防护授权配额有限，选择管理中心后，可查看到当前管理中心剩余的云服务器防护授权数和防篡改授权数。若剩余数量不满足您的需求，请更换管理中心。

    云服务器防护授权数

    主机安全防护配额，一个授权可防护一台云服务器。

    云服务器防护是主机安全服务提供的提升主机整体安全性的核心功能，必须购买该授权才可使用主机安全服务，购买数量最少为 1。

    防篡改授权数

    网页防篡改配额，一个授权可防护一台云服务器。

    网页防篡改是主机安全服务提供的附加增值功能，可根据需要选购，购买数量最少为 0。首次购买主机安全服务时，防篡改授权不可单独购买，须同时购买云服务器防护授权。若首次购买服务时未购买防篡改授权，可在后续升级配额时单独购买。

    说明

    同一个云服务器使用的防篡改授权必须与防护授权在同一个管理中心，其防篡改功能才能生效。

    购买时长

    选择服务购买时长。
    仅支持包年包月购买方式。

    自动续费

    设置是否开启自动续费。
    开启后，服务到期时将自动续费，避免防护中断。自动续费时长与购买时长一致。

    服务协议

    须阅读并同意《服务条款》。必须勾选。

  5. 点击立即购买，进入订单确认页面。

    说明

    若未开启计费对接，则点击确定，完成主机安全服务开通，无需进行支付。

  6. 订单确认无误后，点击确认支付进行支付，完成购买。

    购买成功后，进入主机安全服务的系统管理页面。
