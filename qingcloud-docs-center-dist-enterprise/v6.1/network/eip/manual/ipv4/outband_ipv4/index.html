外部绑定 IPv4 EIP

更新时间：2024-03-07 19:06:53
PDF

本文介绍如何将外部绑定的 EIP 绑定到云服务器、NAT 网关、负载均衡器等云资源上，实现云资源与公网的连接。

背景信息

外部绑定是指在云资源外部通过网络地址转换（NAT）方式将 EIP 地址转换成用户私有网络地址或基础网络地址，从而实现云资源与公网互通。通过外部绑定方式分配 EIP 给云资源后，便可实现云资源公网访问，无须再做额外配置。

前提条件

  • 已申请外部绑定的 EIP。具体操作，请参见申请 EIP。

  • 已创建需要绑定的云资源。

  • 需要绑定的云资源与 EIP 在同一个区域。

约束与限制

  • 绑定 EIP 会给云服务器或负载均衡器增加默认安全组。

  • 私有网络内的负载均衡器或云服务器绑定了 EIP 后，外网访问将使用 EIP 作为进出网关。如果同时配置了 VPC 的端口转发规则，端口转发规则将不再有效。

操作步骤

  1. 登录管理控制台。

  2. 在顶部菜单栏中选择产品与服务 > 网络服务 > 弹性公网 EIP，进入公网 IPv4 页面。

  3. 在 EIP 列表，找到需要分配的外部绑定 EIP，右键点击 EIP，弹出快捷菜单。

  4. 根据需要绑定的资源类型，点击对应的操作，然后选择需要绑定的资源，点击提交，即可完成绑定。

    • 点击分配到云服务器，在弹出的窗口中选择需要绑定的云服务器，将当前 EIP 与所选云服务器绑定。

    • 点击绑定到 VPC 网络，在弹出的窗口中选择需要绑定的 VPC 网络，将当前 EIP 与所选 VPC 网络绑定。

    • 点击分配到负载均衡器，在弹出的窗口中选择需要绑定的负载均衡器，将当前 EIP 与所选负载均衡器绑定。

    • 点击关联到 NAT 网关，在弹出的窗口中选择需要绑定的 NAT 网关，将当前 EIP 与所选 NAT 网关绑定。
