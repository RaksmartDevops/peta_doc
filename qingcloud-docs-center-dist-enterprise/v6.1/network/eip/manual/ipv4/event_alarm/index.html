配置事件告警

弹性公网 EIP 支持状态变更类的事件告警，如解绑公网 EIP。通过设置事件告警策略，您可以在告警事件触发后及时接收、定位并处理问题。

绑定事件告警策略

说明

  • 只有绑定了资源的 EIP 才能绑定事件告警策略。

  • 一个 EIP 实例最多只能绑定一个事件告警策略。

  1. 登录管理控制台。

  2. 在顶部菜单栏中选择产品与服务 > 网络服务 > 弹性公网 EIP，进入公网 IPv4 页面。

  3. 在 EIP 列表，右键点击需要绑定的告警策略的 EIP，在弹出的快捷菜单中，选择告警策略 > 绑定事件告警策略。

  4. 在弹出的窗口中，选择需要绑定的告警策略，点击提交。

    若当前未创建事件告警策略，则点击创建事件告警策略进行创建（推荐），或者点击管理告警策略前往云监控 CloudSat 控制台进行创建。策略设置说明，请参见设置事件告警策略。

  5. 绑定告警策略后，当告警触发，则可以接收告警通知以及查看告警数据。

创建事件告警策略

您可以在绑定事件告警策略时进行创建，也可以前往云监控 CloudSat 控制台进行创建。以下为在云监控 CloudSat 控制台创建的操作步骤。

  1. 登录管理控制台。

  2. 在顶部菜单栏中，选择产品与服务 > 运维与监控 > 云监控 CloudSat。

  3. 在左侧导航栏，点击告警服务。

  4. 在平台告警策略页面，点击创建，弹出创建告警策略页面。

  5. 设置告警参数。

    参数名称 说明

    名称

    事件告警名称。自定义

    对象范围

    保持默认。

    告警类型

    选择事件告警。

    资源类型

    选择公网 IP

    监控周期

    支持 1 分钟或 5 分钟。
    1 分钟粒度需要付费，5 分钟粒度免费。

  6. 点击下一步，设置告警规则。

    参数名称 说明

    告警类型

    公网 IP 当前仅支持状态变更类告警。

    事件名称

    选择需要告警的事件。当前仅支持解绑公网 IP。

    触发条件

    • 立即触发：一旦事件发生则告警。

    • 累计触发：在一段时间内，事件累计发生若干次，才进行告警。需要进行设置。

    告警级别

    • 高：通过邮件、短信、微信和 webhook 进行告警通知。

    • 低：通过邮件和 webhook 进行告警通知。

  7. 可选：点击添加规则，可添加多条规则。

    默认情况下，任何一条规则满足条件时都会触发告警。

  8. 点击下一步，设置告警行为。

    参数名称 说明

    发送通知

    触发告警时是否发送通知。建议选择是，否则无法及时发现告警。

    通知列表

    需要发送通知时，才进行设置。
    选择告警通知的接收人。可点击新列表新增接收人。

  9. 点击提交。

查看事件告警记录

说明

只有已分配的 EIP 才支持在 EIP 控制台查看告警记录，若 EIP 已与资源解绑，您可以在云监控 CloudSat 控制台查看。

通过 EIP 控制台查看

若您已经为 EIP 实例绑定事件告警策略，当告警触发后，您可以查看到告警记录。

  1. 登录管理控制台。

  2. 在顶部菜单栏中选择产品与服务 > 网络服务 > 弹性公网 EIP，进入公网 IPv4 页面。

  3. 在 EIP 列表，点击 EIP 的 ID 号，进入 EIP 详情页。

  4. 点击告警页签，然后点击事件告警，便可查看到事件告警策略及告警历史记录。

通过 CloudSat 控制台查看

  1. 登录管理控制台。

  2. 在顶部菜单栏中，选择产品与服务 > 运维与监控 > 云监控 CloudSat。

  3. 在左侧导航栏，点击事件监控，进入事件监控页面。

  4. 在事件监控页面，可通过设置筛选条件，查看 EIP 的事件记录。

解绑事件告警策略

当您不再需要对 EIP 实例进行事件告警时，可解绑告警策略。

  1. 登录管理控制台。

  2. 在顶部菜单栏中选择产品与服务 > 网络服务 > 弹性公网 EIP，进入公网 IPv4 页面。

  3. 在 EIP 列表，右键点击需要解绑的告警策略的 EIP，在弹出的快捷菜单中，选择告警策略 > 绑定指标告警策略。

  4. 在弹出的窗口中，选择需要解绑的告警策略，点击提交。
