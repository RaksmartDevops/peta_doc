步骤一：创建集群

更新时间：2024-03-07 19:06:53
PDF
本页目录
  • 前提条件
  • 操作步骤

前提条件

  • 已获取管理控制台账号和密码。

  • 已完成个人实名认证且账户余额大于 0 元。

操作步骤

  1. 登录管理控制平台，点击产品与服务 > 计算 > 弹性高性能计算 EHPC， 进入集群管理页面。

  2. 点击创建集群，进入创建 EHPC 集群页面。

  3. 根据需要配置相关参数。参数说明，如下所示。

    参数 说明

    付费类型

    支持包年包月和按需计费。

    • 包年包月：按照订单的购买周期结算。

    • 按需计费：按照实际使用时长计费。

    购买周期

    当选择包年包月时，可设置购买周期。支持 1 个月、3 个月、6 个月、12 个月、3 年以及 5 年。

    自动续约

    当选择包年包月时，可设置自动续约。账户余额充足时，自动续约周期和您购买的周期保持一致，比如：购买一个月，自动续约将续约一个月。

    部署方式

    计算、调度管理和登录节点分离部署。

    • 标准计算：可配置登录节点、管控节点和计算节点；

    • 精简计算：可配置登录节点和计算节点。

    计算节点类型

    通用计算节点：具有 HPC VM 计算、HPC 物理机计算两种计算节点类型，按机器配置支持独立 CPU 计算节点和 GPU 计算节点。

    • HPC VM 计算：根据物理机配置进行设置，一般设置为 CPU 支持 2~32 核，内存对应不同的 CPU 核数支持 4G ~ 160G。

    • HPC BM 计算：提供完整的物理机配置，如 64C 380G 等。

    • HPC GPU 计算：根据 CPU 核心数、内存和 GPU 卡数进行配置，CPU 支持 8 ~ 64 核，内存支持 8G ~ 520G，显卡支持 1 卡 ~ 8卡。

    网络类型

    支持使用私有网络或基础网络。
    若选择使用私有网络，需配置 VPC 网络和私用网络两部分。
    若选择使用基础网络，只需配置基础网络即可。

    VPC网络

    网络类型选择私有网络时，需配置该参数。用来管理相关网络服务，若无 VPC 网络，需先通过关联服务进行创建。可通过 VPC 网络绑定公网 IP 的形式，使得 EHPC 集群中所有节点均可访问互联网。

    说明

    EHPC 集群计算节点为物理机计算节点（裸金属 BM）时，不支持加入 VPC 网络，仅支持加入云平台基础网络。

    私有网络

    网络类型选择私有网络时，需配置该参数。选择已创建的私有网络，若无私有网络，需先通过关联服务进行创建。

    基础网络

    网络类型选择基础网络时，需配置该参数，选择平台内已有的基础网络。

    节点 IP

    节点的 IP 地址，可通过自动分配或手动指定的方式进行获取

    共享文件目录

    专属集群的文件存储，选择共享目录。若未创建共享目录，可点击创建新的共享目录，创建新的共享目录。
    共享目录是集群访问的存储目录，指定存储空间。

  4. 点击下一步，进入系统配置页面，在当前页面用户可查看基本参数，并进行管理配置。

    参数说明如下：

    类别 参数 说明

    基础参数

    镜像类型

    镜像的类型，默认为 CentOS 7.6.
    若为物理机计算节点（即裸金属 BM），默认为 Ubuntu 20.04

    调度器

    默认为 Slurm。

    队列规格

    使用系统默认队列规格。

    域账号服务

    默认为 Idap。

    管理配置

    集群名称

    创建集群的名称，用户自定义。

    集群描述

    选填项，集群的相关描述信息，用户自定义。

    登录方式

    使用密码登录。

    用户名

    登录节点的用户名，默认为 admin。

    设置密码

    登录节点的密码。支持用户自定义或自动生成。

  5. 配置完成后，用户可以在右侧窗口的配置预览中，查看当前 EHPC 集群的相关配置，包括集群名称、可用区、登录节点配置、管控节点配置、计算节点配置、NAS挂载点、调度器相关信息。

  6. 点击确定，完成集群创建操作。

    页面自动返回集群列表页面，若集群状态为活跃，则说明创建集群成功。整个创建过程大约需要 5 分钟，请耐心等待。
