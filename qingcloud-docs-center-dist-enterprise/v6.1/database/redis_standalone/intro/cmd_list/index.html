注释与说明

为便于阅读和内容表达，本文的表格约定使用下述注释：

  • ✓ 表示支持该命令。

  • ✕ 表示不支持该命令。

  • ⎻ 表示在原生 Redis 的该版本下，该命令尚未开始支持。

  • config 表示支持该命令，但需要在控制台修改控制参数支持。
