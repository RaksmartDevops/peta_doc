本章节用于介绍如何关闭云服务。

== 前提条件

* 已获取管理控制台的账号和密码。
* 已创建云服务器，且云服务状态为``运行中``。

== 操作步骤

. 选择 *产品与服务* > *计算* > *云服务器*，进入**云服务器列表**页面。

. 勾选待关机的云服务器，点击云服务列表顶部的**关机**按钮。
+
image::/images/cloud_service/compute/vm/manual_close_vm_1.png[]

. 在弹出的提示窗口中，若确定无误，点击**立即关机**即可。
+

include::../../../../../../../_components/attention.adoc[]
* 待关机的云服务器需确定无业务进行。
* **强制关机**的效果等同于关闭计算机电源，可能会导致数据丢失或文件系统损坏，请仅在云服务器无法正常关机的情况下使用。
