== 基本信息

=== Path
/api/?action=Boss2DescribeNfEmailSettings

=== Method
POST

=== 接口描述
当没有配置时，data 值为 null


== 请求参数

*Headers*

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]
----
{
   "action": "Boss2DescribeNfEmailSettings"
}
----
