== 基本信息

=== Path
/api/?action=Boss2DisableTOTP

=== Method
POST

=== 接口描述
响应数据：

|===
| 响应参数 | 类型 | 描述 | 备注

| id
| Integer
| 用户id
|
|===

|===

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]

{
   "action": "Boss2DisableTOTP",
   "id": 1
}
----
