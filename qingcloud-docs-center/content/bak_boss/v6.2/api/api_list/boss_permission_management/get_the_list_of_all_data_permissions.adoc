== 基本信息

=== Path
/api/?action=Boss2DescribePerms

=== Method
POST

=== 接口描述
获取全部可配置的数据权限信息，用于角色数据权限的配置。
注意：
结果中的id用于后端使用，前端无需进行展示。


== 请求参数

*Headers*

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]
----
{
   "action": "Boss2DescribePerms"
}
----
