== 基本信息

=== Path
/api/?action=Boss2SupervisorDeleteOperationalLogs

=== Method
POST

=== 接口描述
此接口用于超管删除运维日志
请求参数说明

|===
| 字段 | 类型 | 是否必须 | 说明 | 备注

| action
| string
| 是
| Boss2SupervisorDeleteOperationalLogs
|

| operational_log_ids
| array
| 是
| 需要删除的日志id组成的数组
|
|===


== 请求参数

*Headers*

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]
----
{
   "action": "Boss2SupervisorDeleteOperationalLogs",
   "operational_log_ids": [
      2,
      3
   ]
}
----

== 返回数据

[,javascript]
----
{
   "operational_log_ids": [
      3,
      2
   ],
   "ret_code": 0,
   "action": "Boss2SupervisorDeleteOperationalLogsResponse"
}
----
