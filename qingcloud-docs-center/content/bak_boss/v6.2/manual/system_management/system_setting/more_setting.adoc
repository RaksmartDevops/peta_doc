高级设置包括重建缓存，清空任务队列，设置数据留存时间以及重启 BOSS 系统等操作。一般情况下无需设置。



管理员成功登录 BOSS 后，点击**系统管理** > **系统设置** > **高级设置**，进入**高级设置**页面，根据实际需求，点击相应的标签进行设置。

image::/images/boss/manual/system_mgt/more_setting.png[more_setting,90%]


== 重建缓存

当{platform_name_cn}的计算节点或管理节点发生变化，如扩容等场景下，系统默认 2 小时更新系统配置，若需快速更新，可**点击重建**，一分钟后，系统配置即可生效。

== 任务队列

当异步任务队列积压时，可选择手动清空。BOSS 建议先调查清楚任务队列积压的原因并解决后，再执行清空队列的操作。

== 数据留存时间

BOSS 默认不设置审计、告警以及监控数据的留存时间，管理人员可根据实际需求，自行设置。

== 系统重启

为管理人员提供重启 BOSS 服务的入口。
