== 场景描述

该任务指导管理员进入公告通知页面，查看云计算服务平台（Console）和云计算运维管理平台（BOSS）的公告通知。

== 功能描述

公告通知页面展示了公告颜色、公告时限、公告展示、公告内容等。

通过该页面可以查看、编辑公告通知。

== 前提条件

已获取 BOSS 的登录账号和密码。

== 操作步骤

. 使用浏览器登录 BOSS。
. 选择 *系统管理* > *云平台设置* > *公告通知*，进入公告通知页面。
+
image::/images/boss/manual/system_mgt/notice_1.png[]

. 点击 *编辑*，可分别对云计算服务平台（Console）和云计算运维管理平台（BOSS）的公告进行编辑。
+
image::/images/boss/manual/system_mgt/notice_2.png[]
+
参数说明如下：
+	
[cols="2*", options="header"]
|===
|参数类型
|参数说明

|公告内容|填写需要展示的公告内容。

|公告颜色|设置公告的颜色，可选择红色和黄色。

|公告展示|选择 *启用*，则展示公告；选择 *禁用*，则不展示公告。

|公告时限|可选项。选择公告的展示时间。
|===

. 编辑完成后，点击 *确认发布*。
+

include::../../../../../_components/note.adoc[]
若未配置 *公告时限*，发布后该公告会立即生效。
