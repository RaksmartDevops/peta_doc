== 功能描述

BOSS 支持设置回收站资源保留时间与登录保护。

* 回收站资源保留时间针对 Global 下所有 Zone 生效，默认保留 `2小时`，用户可根据需求进行修改。
* 登录保护中，用户可统一配置账号在限定时间内连续登录失败多次会被临时锁定一段时间，以避免账号被暴力破解盗用，从而造成损失。

== 前提条件

已获取 BOSS 的登录账号和密码。

== 操作步骤

. 使用浏览器登录 BOSS。

. 点击 **系统管理** > **云平台设置** > **云平台全局设置**，进入**云平台全局设置**页面。
+
image::/images/boss/manual/system_mgt/global_setting.png[]

. 根据实际需求与页面提示信息，修改**回收站资源保留时间**，修改**登录设置**相关参数。
+
include::../../../../../_components/note.adoc[]

**回收站资源保留时间**修改后，针对之后进入回收站的数据生效。修改配置之前已经进入回收站的数据，仍然按照之前设置的参数保留。

include::../../../../../_components/table_footer.adoc[]


. 确认修改无误后，点击**应用修改**。
