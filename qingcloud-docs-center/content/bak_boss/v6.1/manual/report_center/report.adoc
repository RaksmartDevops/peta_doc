== 场景描述

该任务指导用户进入 *报表管理中心* 页面，查看报表数据并下载到本地。

== 功能描述

报表管理中心是对本管理平台所有资源相关数据导出报表的统一管理，可以提供下载等功能，采用异步导出的方式，让用户导出报表过程中不需要等待。

== 前提条件

已获取 BOSS 的登录账号和密码。

== 操作步骤

. 使用浏览器登录 BOSS。
. 在左侧导航栏点击 *报表管理中心*，进入 *报表管理中心* 页面，可查看所有云用户的状态、模块、文件名称等相关信息。
+
image::/images/boss/manual/report_center/report_operation.png[]

. 点击页面上方状态，可展示该状态所有报表。

=== 导出报表

include::../../../../_components/note.adoc[]
只有报表状态为 *已完成* 时，可正常下载。

include::../../../../_components/table_footer.adoc[]

. 在 *报表管理中心* 页面搜索框中输入文件名称，搜索待下载的报表，按 *Enter*.
. 点击待下载报表的文件名称，文件下载到本地。
