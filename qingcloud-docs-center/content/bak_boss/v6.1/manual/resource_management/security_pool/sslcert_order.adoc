用户可以上传自己本地的 SSL 证书，也可在{platform_name_cn}申请购买 SSL 证书。在{platform_name_cn}申请购买的 SSL 证书会生成对应的订单呈现在 *云计算运维管理平台* 上由管理员查看。

== 场景描述

该任务指导管理员进入 *安全：SSL 证书订单* 页面，查看{platform_name_cn}用户申请购买的 SSL 证书的订单列表信息。

== 功能描述

*安全：SSL 证书订单* 页面集中展示云环境中用户申请购买的所有 SSL 证书的订单列表、证书订单号、状态、证书品牌、证书名称、证书类型、域名、关联证书 ID、用户邮箱/ID、创建时间以及支付时间信息。

== 前提条件

已获取 BOSS 的登录账号和密码。

== 操作步骤

. 使用浏览器登录 BOSS。
. 选择 *资源管理* > *安全* > *SSL 证书订单*，进入 *安全：SSL 证书订单* 页面，可查看 SSL 证书订单列表和详情。
+
image::/images/boss/manual/resource_mgt/sslcert_order_operation.png[]

. 点击 SSL 证书订单的证书订单号，进入订单详情页面，可查看 SSL 证书订单的详细信息。
+
在 *证书详情* 页签，可查看证书的公司详情和联系人信息。
+
在 *确认函* 页签，可查看订单购买确认函附件信息。

=== 吊销证书

include::../../../../../_components/note.adoc[]
证书吊销之后证书失效，该操作不可撤回，请谨慎操作。

include::../../../../../_components/table_footer.adoc[]

. 在订单详情页面右上角点击 *吊销证书*。
+
弹出 *吊销证书* 对话框。

. 输入操作原因，点击 *确认*。

=== 取消订单

include::../../../../../_components/note.adoc[]
订单取消之后不可撤回，请谨慎操作。

include::../../../../../_components/table_footer.adoc[]

. 在订单详情页面右上角点击 *取消订单*。
+
弹出 *取消订单* 对话框。

. 输入操作原因，点击 *取消*。
