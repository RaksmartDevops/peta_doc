== 背景信息

SaaS 应用是{platform_name_cn}将服务商提供的在线应用软件作为商品。用户无需购买独立的云资源，只需要购买 SaaS 应用即可登录到指定的网站使用商品。

// 为使用户能够使用更多更专业的功能，而这些功能可由专业的服务提供商提供。同时，青云能够为 IT 产业提供 IaaS 平台，帮助专业服务提供商拓展业务。青云以开发应用的形式，让第三方公司为青云用户提供各种专业服务。服务提供商仅需提供服务网页，整合到 QingCloud iframe 里面供用户使用。加载 iframe 时，用户授权信息会发送到对应的网址。服务提供商拿到授权信息就可以通过 QingCloud API 获取用户资源信息，为用户提供各种服务。

// 服务提供商提 URL 整合到青云界面，青云不需要调用服务提供商的 API 就能让用户使用其服务。

// 提供一致的用户体验。用户所有操作都在青云用户界面完成，不需要登录到服务提供商网站。

// 最大程度保护用户信息和资源，服务提供商需要被用户授权后才能通过青云 API 获取用户相关信息。

// 为增强用户体验，青云提供用于内联网的域名和公网服务的反向代理加速内网访问。

== 场景描述

该任务指导管理员进入 SaaS 应用页面，查看 SaaS 应用列表详情和预览应用。

== 功能描述

SaaS 应用列表页面罗列了{platform_name_cn}上用户上传的所有 SaaS 应用的状态、应用名称/ID、可见范围、开发者/ID 以及创建时间等信息。

== 前提条件

已获取 BOSS 的登录账号和密码。

== 操作步骤

. 使用浏览器登录 BOSS。
. 在左侧导航栏选择 *资源管理* > *AppCenter* > *SaaS 应用*，可查看所有状态的 SaaS 应用列表。
+
image::/images/boss/manual/resource_mgt/saas_apps_operation.png[]

. 点击 SaaS 应用的名称，进入 SaaS 应用的详情页面。
+
在 *配置* 页签，可查看当前 SaaS 应用的配置信息（如应用 URL、应用密钥）、支持区域以及权限信息（如访问授权、编辑权限和访问权限）。
+
在 *价格* 页签，可查看当前 SaaS 应用按时长付费价格的折扣、价格、开始时间和结束时间。
+
在 *条款及文档* 页签，可查看当前 SaaS 应用的服务条款和使用说明。
+
在 *应用截图* 页签，可查看当前 SaaS 应用的大致截图信息。
+
在 *审核记录* 页签，可查看当前 SaaS 应用所有审核记录的审核结果、审核人/邮箱、角色、审核时间以及发送审核通知时间等信息。
+
image::/images/boss/manual/resource_mgt/saas_apps_info.png[]

=== 预览应用

include::../../../../../_components/note.adoc[]
仅拥有 global 权限的管理员可预览应用。

include::../../../../../_components/table_footer.adoc[]

在待预览 SaaS 应用所在行末尾点击image:/images/boss/manual/icon.png[]，点击 *预览应用*。

=== 审核应用

include::../../../../../_components/note.adoc[]

SaaS 应用状态为 *待审核* 时，可执行应用审核操作。

include::../../../../../_components/table_footer.adoc[]


. 在待审核 SaaS 应用所在行末尾点击image:/images/boss/manual/icon.png[]，点击 *应用审核*。
+
弹出 *应用审核* 对话框。

. 勾选 *技术审核* 或者 *渠道审核*。并输入审核反馈。
+

include::../../../../../_components/note.adoc[]
渠道审核：主要审核应用信息、价格信息以及资质信息等。

技术审核：主要审核应用是否能正常运行，技术审核通过前需做镜像迁移，否则不能技术进行技术审核，且技术审核按钮不能点击。

include::../../../../../_components/table_footer.adoc[]

. 点击 *审核通过* 或者 *审核拒绝*，完成审核操作。

=== 暂停应用

include::../../../../../_components/note.adoc[]
SaaS 应用状态为 *已上架* 时，可在执行暂停应用操作。

include::../../../../../_components/table_footer.adoc[]

. 在待预览 SaaS 应用所在行末尾点击image:/images/boss/manual/icon.png[]，点击 *在应用市场查看*。
+
弹出 *暂停应用* 对话框。

. 输入操作原因吗，点击 *暂停*。

=== 在应用市场查看应用


SaaS 应用状态为 *已上架* 时，可在应用市场查看该应用。
