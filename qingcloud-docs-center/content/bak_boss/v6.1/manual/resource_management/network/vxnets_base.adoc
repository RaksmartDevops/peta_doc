== 背景信息

{platform_name_cn}基础网络，是一个可以与企业已有网络三层可达的网络。该网络可以理解为传统数据中心网络中心的一个（或多个）VLAN 段，这些网段专门用于承载{platform_name_cn}上的虚拟机。但是与传统数据中心不同的是，基础网络网关一般由{platform_name_cn}以 SDN 形式实现，{platform_name_cn}也支持将基础网络网关设置在物理交换机上。

基础网络的优点是简单。在{platform_name_cn}配置完毕后，{platform_name_cn}用户无需做任何配置与管理，即可直接使用该网络。基础网络是全局网络，不同租户在基础网络同一网段创建的虚拟机在二层互通，其安全保障需要依靠防火墙（Security Group）来实现。{platform_name_cn}基础网络一般是采用 VXLAN 技术所搭建，也支持 VLAN 形式。

== 场景描述

该任务指导管理员进入基础网络页面，查看基础网络相关信息，执行创建基础网络和修改基础网络操作。

== 功能描述

基础网络也是私有网络的一种形式，但是其安全性、私密性要低于私有网络。

基础网络列表展示当前云环境中所有的基础网络的名称及 ID、IP 空闲数、IP 总数、VLAN 号、工作模式、物理网卡 ID、IP 网段、创建时间信息。

== 前提条件

已获取 BOSS 的登录账号和密码。

== 操作步骤

. 使用浏览器登录 BOSS。
. 在左侧导航栏选择 *资源管理* > *网络* > *基础网络*，在左上角面板上选择区域或可用区，可查看基础网路列表及相关信息。
+
image::/images/boss/manual/resource_mgt/vxnets_base_operation.png[]

. 点击基础网络的名称，可查看基础网络详细信息。

* 在 *关联资源* 页签，可查看私有网络中所有的主机状态、名称及 ID、资源类型、IP 地址、DHCP 配置信息，支持在多个主机资源中搜索。

* 在 *网络监控* 页签，可查看指定时间段内的包转发率、带宽、实时监控数据。

* 在 *分布式虚拟路由器* 页签，可查看该私有网络的分布式路由器的角色、所在计算节点、虚拟主机 ID、网络地址信息。

* 在 *特权用户* 页签，可查看该基础网络被使用的特权用户的 ID、名称和邮箱。

=== 创建基础网络

. 在 *基础网络* 页面右上角，点击 *创建*。
+
弹出 *创建基础网络* 对话框。

. 根据提示信息输入参数。
+
	
[cols="2,8a", options="header"]
|===
|参数
|参数说明

|名称
|为所创建的基础网络填写一个名称。

|隔离模式
|已根据您的环境信息自动配置隔离模式。

|VLAN ID/VXLAN ID
|* 隔离模式为 VLA 时：VLAN ID 必填，由用户直接输入，取值范围：2～4094。
* 隔离模式为 VLA 时：VXLAN ID 不支持用户输入，由系统自动分配，取值范围：0-16777215。

|IP 网段
|IP 网段必须在地址段 10.0.0.0/8 内。

* 若隔离模式为 VLAN，该处为 VLAN 网络的 IP 地址段。
* 若隔离模式为 VXLAN，该处为 VXLAN 网络的 IP 地址段。

|选择节点
|隔离模式为 VXLAN 时，必须从计算节点列表中选择一个节点。

|高级功能
|勾选高级功能，可设置工作模式、管理网络 IP、网关 IP 和 DHCP 服务。

|工作模式
|* 普通模式：最常用的普通虚拟机网络，DHCP 和网关等由{platform_name_cn}管理，支持 VLAN 模式或者 VXLAN 模式。
* 直通模式：近似物理机的网络性能，DHCP 和网关等由物理交换机管理，可以与计算节点网络互通，仅支持 VLAN 模式。

|物理网卡
|隔离模式为 VALN 时，此处输入物理网卡地址，可选填。

|管理网络 IP
|隔离模式为 VXALN 时，此处输入管理网络 IP 地址，必填。用于与本基础网络与其他基础网络间的通信。

|网关 IP
|输入网关 IP 地址，选填。

|DHCP 服务
|DHCP 服务器 IP 和网关 IP 不在起始 IP 和终止 IP 范围内。开启之后可自定义设置 DHCP 服务器 IP、DHCP 起始 IP 地址和 DHCP 结束 IP 地址。
|===

. 点击 *创建*。

=== 修改基础网络

. 在 *基础网络* 页面搜索框中搜索待修改基础网络 ID 或名称，按 *Enter*，找到待修改基础网络。
. 在待修改基础网络所在行点击 image:/images/boss/manual/icon.png[]，点击 *修改*。
+
弹出 *修改基础网络* 对话框。

. 修改基础网络名称，输入备注和操作原因。
. 点击 *修改*。
