== 基本信息

=== Path
/api/?action=Boss2CreateRole

=== Method
POST

=== 接口描述
用于创建BOSS2系统中的角色，将会返回角色id和角色名称。
响应数据：

|===
| 参数名称 | 类型 | 描述 | 备注

| id
| String
| 角色id
|

| name
| String
| 角色名称
|
|===


== 请求参数

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===
