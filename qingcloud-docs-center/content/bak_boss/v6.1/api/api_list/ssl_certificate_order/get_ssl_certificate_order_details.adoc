== 基本信息

=== Path

=== Method
POST

== 请求参数

*Headers*

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]
----
{
   "action": "Boss2SSLCertShopGetOrderDetail",
   "order_id": "oid-2c3d2f4e9f63"
}
----
