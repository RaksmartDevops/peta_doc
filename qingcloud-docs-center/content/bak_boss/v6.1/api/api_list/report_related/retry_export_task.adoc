== 基本信息

=== Path
/api/?action=Boss2RetryReportTask

=== Method
POST

=== 接口描述
响应数据：

|===
| 参数名称 | 类型 | 描述 | 备注

| token
| String
| 口令
|

| valid_fields
| Array
| 有效字段声明列表
|
|===


== 请求参数

*Headers*

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]
----
{
   "action": "Boss2RetryReportTask",
   "token": "680743b8-5ce9-47b0-b1a1-99920737523c"
}
----
