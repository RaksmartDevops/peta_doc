== 基本信息

=== Path
/api/?action=Boss2DescribeMySalesDept

=== Method
POST

=== 接口描述
获取当前用户可管理的部门列表。
返回数据：

|===
| 字段 | 说明 | 备注

| ka_type
| 部门类型id
|

| ka_type_name
| 部门类型名称
|

| leader
| 部门负责人id
|

| leader_email
| 部门负责人邮箱
|

| leader_name
| 部门负责人姓名
|

| sales_dept_id
| 部门id
|

| sales_dept_name
| 部门名称
|

| sales_count
| 部门销售数量
|

| level
a|
部门层级

* 1 - 一级部门
* 2 - 二级部门
|

| parent_id
| 父级部门id
|
|===


== 请求参数

*Headers*

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]
----
{
    "action": "Boss2DescribeMySalesDept",
    "sales_dept_id": [7],  // 使用部门id进行过滤
    "sales_dept_name": ["测试部门1"],  // 使用部门名称进行过滤
    "ka_type": [1, 2, 99]   // 使用部门类型id进行过滤
}
----
