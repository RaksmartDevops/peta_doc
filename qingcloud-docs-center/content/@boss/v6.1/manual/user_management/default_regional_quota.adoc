== 背景信息

云用户在新注册账户后，会使用到 *云计算运维管理平台* 中的各种资源，系统需要提前为用户设置资源默认配额。

== 场景描述

该任务指导管理员进入 *默认区域配额* 页面，为云用户设置或修改默认配额。

== 功能描述

该页面可设置或修改计算、存储、网络、数据与缓存、大数据平台、计费、集群、安全、弹性容器实例及其他资源配额。

== 前提条件

已获取 BOSS 的登录账号和密码。

== 操作步骤

. 登录 *云计算运维管理平台* 。
. 在左侧导航栏选择 *用户管理* > *默认区域配额*，进入 *默认区域配额* 页面。
+
image::/images/boss/manual/user_management/default_quota_operation.png[]

=== 修改资源默认配额

include::../../../../_components/note.adoc[]
* 默认配额不得超过公司全部资源池数量。

* 默认配额即 Console 用户的初始配额，该修改仅对新建用户的初始配额有效。

* 修改默认配额时，为保证修改配额生效，请在一个产品类型的配额修改完之后，点击保存后再修改下一个产品类型的配额。

include::../../../../_components/table_footer.adoc[]

. 在 *默认区域配额* 页面选择区域。
. 点击 *修改配额*，修改计算、存储、网络、AppCenter 集群等资源的配额信息。
+
image::/images/boss/manual/user_management/default_quota_01.png[]

. 修改完成后，点击 *保存*。
