== 基本信息

=== Path
/api/?action=Boss2RunSchedulerTask

=== Method
POST

=== 接口描述
立即执行指定ID的定时器对应的任务。
该接口仅可执行当前用户所拥有的定时器任务。


== 请求参数

*Headers*

[cols="3*", options="header"]

|===
| 参数名称 | 参数值 | 是否必须

| Content-Type
| application/json
| 是
|===

*Body*

[,javascript]
----
{
  "action": "Boss2RunSchedulerTask",
  "id": 41 // [必须]指定要执行的定时器id
}
----

== 返回数据

[,javascript]
----
{
   "id": 41,
   "ret_code": 0,
   "action": "Boss2RunSchedulerTaskResponse"
}
----
