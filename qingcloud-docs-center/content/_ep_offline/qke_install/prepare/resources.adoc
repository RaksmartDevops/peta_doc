== Harbor 应用安装包

Harbor 用于创建私有镜像仓库，然后需要将 qke_harbor.tar.gz 上传到创建的私有镜像仓库中。

include::../../../_components/note.adoc[]
如果客户有自己的镜像仓库，也可以不用安装 Harbor 应用，请确保云平台环境与客户的私有镜像仓库网络互通即可。
include::../../../_components/table_footer.adoc[]

== QKE 资源包

仅 QKE v3.0.0 版本需要下载如下 QKE 依赖资源。

创建 QKE v3.0.0 集群时需要依赖 Docker 镜像和 QKE 集群安装包，所以在安装 QKE 应用时需要在环境中配置好依赖的资源。

下载路径：`\ftp://ins.qingcloud.com/k8s/QKE/v3.0.0/amd64/`。

[cols='1,2'] 
|===
| 资源 | 说明 

| qke_harbor.tar.gz
| 主要包含 Docker 镜像和推送 Docker 镜像到 Harbor 镜像仓库的脚本。

| qke-cli-v3.0.0.tar.gz
| 用于安装 QKE CLI 工具。

| qke-installer-v3.0.0.tar.gz
| 用于创建 K8s 集群。

| qke_nginx.tar.gz
| 用于搭建 QKE 专属的文件服务器。
|===
